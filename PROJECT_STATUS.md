## How to Build the GitBook

### Option 1: GitBook.com (Easiest)
1. Create account at gitbook.com
2. Import this directory as a new book
3. GitBook will read SUMMARY.md and build navigation
4. Publish!

### Option 2: GitBook CLI (More Control)
```bash
# Install GitBook CLI
npm install -g gitbook-cli

# Navigate to this directory
cd /Users/annhoward/src/multiverse_admin_handbook

# Initialize and serve locally
gitbook init
gitbook serve

# Build static HTML
gitbook build
```

### Option 3: GitHub Pages
- Push to GitHub repository
- Enable GitHub Pages
- GitBook will auto-build from SUMMARY.md
