# Emergency Response Flowchart

## Is This an Emergency?

### YES - Immediate Danger
Student mentions:
- Suicide plan or intent
- Violence toward others
- Currently unsafe situation
- Psychotic break with risk

**ACTION:**
1. Call 988 or provide crisis resources immediately
2. Do not attempt intervention alone
3. Notify admin IMMEDIATELY
4. Document everything
5. If imminent danger and they won't call: consider calling 911 (use judgment)

---

### MAYBE - Crisis But Not Immediate
Student showing:
- Emotional overwhelm
- Paranoia/delusions (but not acting dangerously)
- Manic behavior
- Major life crisis

**ACTION:**
1. Use grounding language: "Are you safe right now?"
2. Provide crisis resources (988, Crisis Text Line)
3. Do not provide therapy
4. Escalate to admin
5. Document
6. Follow up: "Have you contacted support?"

---

### NO - Challenging But Not Crisis
Student:
- Boundary pushing
- Emotional dysregulation (upset but not unsafe)
- Difficult behavior
- Conflict with peers

**ACTION:**
1. Set clear boundary
2. Document if it's a pattern
3. Redirect to appropriate resources
4. Escalate if it continues
5. Self-care check: How are YOU doing?

---

## Quick Decision Tree

```
Is someone in immediate danger?
├─ YES → Call 988, notify admin, document
└─ NO ↓

Is this a mental health crisis?
├─ YES → Provide crisis resources, escalate to admin
└─ NO ↓

Is this a boundary violation?
├─ FIRST TIME → Set boundary clearly, document
└─ REPEATED → Escalate to admin, consider consequences

Is this neurodivergent communication?
├─ YES → Clarify expectations, accommodate when possible
└─ Unclear → Consult with peers/admin
```
